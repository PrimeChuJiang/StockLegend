# 竞选大师 — 核心规则 v4

> 黑色幽默政治竞选卡牌游戏。
> 核心玩法：素材卡合成竞选广告。复杂度集中在合成环节，其他一切从简。
> 美式互攻桌游，直接对抗。

---

## 1. 一句话描述

你是一个不择手段的竞选经理，通过合成竞选广告来拉票、抹黑对手。选举日数票，票多者胜。

---

## 2. 游戏配置

| 参数 | 值 | 备注 |
|------|-----|------|
| 玩家数 | 2 人 | 验证后可扩展 |
| 选区数 | 5 个 | 每个选区有不同数量的选民和选票价值 |
| 选民总数 | 25 个 | 分布在 5 个选区 |
| 回合数 | 6 回合 | 竞选季最后 6 周 |
| 初始状态 | 所有选民中立 | |

## 3. 选区与选民

### 3.1 选区地图

| 选区 | 选民数 | 选票价值 | 关心议题 |
|------|-------|---------|---------|
| 铁锈镇 | 5 | 3 票 | 就业 |
| 硅谷湾 | 4 | 4 票 | 科技 |
| 农心县 | 6 | 2 票 | 农业 |
| 华尔街 | 3 | 5 票 | 经济 |
| 大学城 | 7 | 3 票 | 教育 |

总选票 = 17 票，**9 票获胜**。

选民多的选区不一定值钱。华尔街只有 3 个选民但值 5 票——兵家必争。

### 3.2 赢者通吃

每个选区独立计算：谁拥有更多选民 → 获得该选区**全部选票**。平局则双方都不得分。

### 3.3 选民属性

| 属性 | 说明 |
|------|------|
| 所在选区 | 固定不动 |
| 归属 | 中立 / 玩家A / 玩家B |
| 忠诚度 | 整数 ≥ 0 |

## 4. 素材卡

### 4.1 四种素材

| 类型 | 说明 | 擅长 |
|------|------|------|
| **政绩卡 RECORD** | 真实数据和执政成果 | 稳定拉票、高效打假 |
| **承诺卡 PROMISE** | 竞选承诺，不一定兑现 | 强力拉票，但有翻车风险 |
| **黑料卡 DIRT** | 对手的丑闻和黑历史 | 攻击对手、高效打假 |
| **煽情卡 EMOTION** | 煽动性情绪素材 | 拉票和攻击都行，万金油 |

### 4.2 素材品质

| 品质 | 基础值 | 牌堆占比 |
|------|-------|---------|
| 普通 ★ | 1 | 50% |
| 劲爆 ★★ | 2 | 35% |
| 核弹级 ★★★ | 3 | 15% |

### 4.3 议题标签

每张素材卡带有一个议题标签（就业/科技/农业/经济/教育/无）。广告投放到匹配议题的选区时效果更强。

### 4.4 牌堆

每种素材 10 张，共 40 张。品质和议题均匀分配。

## 5. 广告合成（核心玩法）

### 5.1 合成规则

消耗 **1 行动点 + 2 张手牌** → 合成一则广告 → 选择投放到哪个选区。

素材的**类型组合**决定广告类型。素材的**品质之和**决定广告强度（2~6）。

### 5.2 广告类型表

| 素材组合 | 广告类型 | 效果类型 |
|---------|---------|---------|
| 政绩 + 政绩 | 执政报告 | 纯拉票 |
| 政绩 + 承诺 | 施政蓝图 | 拉票为主 |
| 政绩 + 煽情 | 感人故事 | 拉票 + 防御 |
| 政绩 + 黑料 | 调查报告 | 攻击为主 + 防御 |
| 承诺 + 承诺 | 空头支票 | 极强拉票，可被打假 |
| 承诺 + 煽情 | 竞选演说 | 强力拉票 |
| 承诺 + 黑料 | 对比广告 | 拉票 + 攻击 |
| 黑料 + 黑料 | 丑闻连环锤 | 纯攻击 |
| 黑料 + 煽情 | 恐惧广告 | 强力攻击 |
| 煽情 + 煽情 | 民粹煽动 | 拉票 + 攻击 |

### 5.3 效果数值

**广告强度 = 两张素材品质之和（2~6）**

**拉票系数表：**

| 广告类型 | 拉票系数 | 攻击系数 |
|---------|---------|---------|
| 空头支票 | 2.5 | 0 |
| 竞选演说 | 2.0 | 0 |
| 执政报告 | 2.0 | 0 |
| 施政蓝图 | 1.5 | 0 |
| 感人故事 | 1.5 | 0 |
| 民粹煽动 | 1.0 | 0.5 |
| 对比广告 | 0.5 | 1.5 |
| 调查报告 | 0 | 1.5 |
| 恐惧广告 | 0 | 2.0 |
| 丑闻连环锤 | 0 | 2.5 |

**拉票影响力 = 强度 × 拉票系数**
**攻击打击力 = 强度 × 攻击系数**

### 5.4 议题匹配加成

广告中任意素材的议题标签匹配目标选区议题：
- 1 张匹配 → 效果 ×1.5
- 2 张匹配 → 效果 ×2.0

### 5.5 拉票结算

拉票影响力在目标选区内分配：
1. 优先影响**中立选民** → 归属你，忠诚度 = 分到的影响力
2. 中立用完后影响**对手选民** → 削忠诚度，归零变中立，剩余归你

影响力在选民间**平均分配**（向下取整）。

### 5.6 攻击结算

攻击打击力降低目标选区中**对手选民**的忠诚度：
- 打击力平均分配到对手在该选区的所有选民
- 忠诚度降到 0 → 变中立

### 5.7 空头支票的风险

空头支票（承诺+承诺）拉票最强，但标记为"可被打假"：
- 被打假 → 效果归零 + 你在该选区已有选民忠诚度 -2
- 未被打假 → 正常生效

---

## 6. 事实核查（打假）

消耗 **1 行动点 + 1~2 张素材卡**。

### 6.1 通用打假

降低目标选区中对手选民的忠诚度。

打假影响力 = 素材品质之和 × 类型加成：
- 政绩卡 ×1.5
- 黑料卡 ×1.5
- 其他 ×1.0

### 6.2 针对空头支票

如果对手本回合在某选区投放了空头支票，你可以针对性打假：
- 空头支票效果归零
- 对手在该选区已有选民忠诚度 -2
- 比通用打假更高效

---

## 7. 声势等级

你控制的选民总数决定声势：

| 选民数 | 等级 | 抽牌数 |
|-------|------|-------|
| 0~3 | Lv.1 | 3 张 |
| 4~7 | Lv.2 | 4 张 |
| 8~12 | Lv.3 | 5 张 |
| 13~17 | Lv.4 | 6 张 |
| 18+ | Lv.5 | 7 张 |

声势只影响抽牌数，不影响广告效果。避免雪球太猛。

---

## 8. 回合流程

```
┌─────────────────────────────────────────────┐
│ 1. 抽牌阶段                                  │
│    · 双方各按声势等级抽牌                      │
├─────────────────────────────────────────────┤
│ 2. 玩家A行动阶段（3 行动点）                  │
│    · 投放广告：1行动点 + 2张素材 → 选选区      │
│    · 事实核查：1行动点 + 1~2张素材 → 打假      │
│    · 用完或主动结束                            │
├─────────────────────────────────────────────┤
│ 3. 玩家B行动阶段（3 行动点）                  │
│    · 同上                                     │
├─────────────────────────────────────────────┤
│ 4. 结算阶段                                  │
│    · 所有有主选民忠诚度 -1                     │
│    · 忠诚度归零 → 变中立                       │
│    · 超出手牌上限则弃到上限                    │
└─────────────────────────────────────────────┘
```

### 8.1 行动点

| 操作 | 消耗 |
|------|------|
| 投放广告 | 1 行动点 + 2 张素材 |
| 事实核查 | 1 行动点 + 1~2 张素材 |

每回合 3 行动点。

### 8.2 手牌

| 参数 | 值 |
|------|-----|
| 手牌上限 | 10 张 |

### 8.3 先后手

每回合先后手交替。第 1 回合玩家A先行动，第 2 回合玩家B先行动，以此类推。

---

## 9. 胜负条件

第 6 回合结算结束后，进行**选举日计票**：

1. 逐选区统计：谁拥有更多选民 → 获得该选区全部选票
2. 平局选区双方不得分
3. 总票数多的玩家获胜（9/17 票即胜）
4. 总票数平局 → 比较控制选民总数，多的获胜

## 10. 一回合示例

```
=== 第 3 回合 ===

选情：你 5票(铁锈镇+农心县) vs 对手 4票(硅谷湾) → 8票未定
你的声势：6 选民 → Lv.2 → 抽 4 张

[抽牌] 抽到：黑料★★★(经济) + 承诺★(教育) + 政绩★★(就业) + 煽情★(无)

[你的行动阶段] 3/3 行动点

  操作1：投放广告 → 华尔街
  黑料★★★(经济) + 煽情★(无) → 恐惧广告，强度 4
  攻击力 = 4 × 2.0 = 8，经济匹配华尔街 ×1.5 = 12
  → 对手在华尔街的 1 个选民被打掉忠诚度，变中立
  行动点 2/3

  操作2：投放广告 → 华尔街
  政绩★★(就业) + 承诺★(教育) → 施政蓝图，强度 3
  拉票 = 3 × 1.5 = 4（无议题匹配）
  → 华尔街 3 个选民中有中立的，影响力 4 分配给中立选民
  行动点 1/3

  操作3：保留行动点（没有好牌了）

[对手行动阶段] ...

[结算] 所有有主选民忠诚度 -1
```

## 12. 扩展方向

1. **事件系统** → 每回合翻事件卡，基于真实竞选闹剧（内容层，框架确认后再填）
2. **辩论机制** → 特殊回合，双方同时出牌对决
3. **十月惊奇** → 最后一回合特殊高影响力事件
4. **背书系统** → NPC 支持（工会、名人、企业）
5. **3~4 人模式** → 多方混战
6. **角色能力** → 不同候选人有被动特殊能力
7. **金主系统** → 资金维度
8. **副总统搭档** → 开局选搭档提供被动加成
