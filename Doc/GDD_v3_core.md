# StockLegend 核心规则 v3 — 自媒体经营模型

> 本文档定义单人最小可玩原型的完整规则。
> 核心玩法：素材合成文章。围绕"写文章"展开所有决策。
> 多人规则在单人验证通过后自然扩展。

---

## 1. 一句话描述

你是一个财经自媒体人，通过撰写不同类型的文章来赚钱、圈粉、打压对手。粉丝越多，你的影响力越大，能写出更强的文章。在有限回合内比谁赚得多。

---

## 2. 游戏配置（最小原型）

| 参数 | 值 | 备注 |
|------|-----|------|
| 散户总数 | 15 个 | 可争夺的粉丝池 |
| 回合数 | 8 回合 | |
| 初始资金 | 0 | 白手起家 |
| 初始散户 | 全部无主 | |
| 玩家数 | 2（玩家 vs AI） | |

---

## 3. 核心循环

```
抽牌（散户越多抽越好）
    ↓
合成文章（核心决策）
    ↓
  ┌──────────┬──────────┬──────────┐
  赚钱文章    圈粉文章    打假文章
  （变现）    （投资）    （攻击）
  ↓           ↓           ↓
  获得金币    获得散户    对手掉粉
  └──────────┴──────────┴──────────┘
    ↓
散户越多 → 影响力越高 → 下回合抽牌更好 → 文章更强……
```

**核心决策：每回合手牌有限，写什么类型的文章？**

- 赚钱文章 = 短期收益（直接拿钱）
- 圈粉文章 = 长期投资（散户多了以后赚更多）
- 打假文章 = 攻击对手（削弱对手的滚雪球）

三者共享同一份资源（素材卡 + 行动点），互斥竞争。

## 4. 素材卡

### 4.1 素材类型

| 类型 | 说明 | 擅长合成 |
|------|------|---------|
| **数据卡 DATA** | 财报、统计、图表 | 赚钱文章、打假文章 |
| **爆料卡 SCOOP** | 内幕消息、泄露文件 | 赚钱文章、圈粉文章 |
| **谣言卡 RUMOR** | 小道消息、论坛传闻 | 圈粉文章 |
| **观点卡 OPINION** | 专家评论、分析师观点 | 赚钱文章、圈粉文章 |

### 4.2 素材品质

每张素材卡有品质等级，影响合成文章的效果：

| 品质 | 基础值 | 牌堆占比 |
|------|-------|---------|
| 普通 ★ | 1 | 50% |
| 优质 ★★ | 2 | 35% |
| 重磅 ★★★ | 3 | 15% |

### 4.3 牌堆构成（最小原型）

每种素材类型各 10 张（共 40 张），品质按上述比例分配：
- 每种类型：5 张普通 + 3 张优质 + 2 张重磅

## 5. 文章合成系统

### 5.1 合成规则

玩家选择 **2 张素材卡** 合成一篇文章。素材的**类型组合**决定文章类型，素材的**品质之和**决定文章强度。

消耗：1 行动点 + 2 张手牌

### 5.2 文章类型表

| 素材组合 | 文章类型 | 主要效果 | 说明 |
|---------|---------|---------|------|
| 数据 + 数据 | 深度研报 | 赚钱 +++ | 最赚钱，但不圈粉 |
| 数据 + 观点 | 专家点评 | 赚钱 ++ 圈粉 + | 稳健收益 |
| 数据 + 爆料 | 调查报道 | 赚钱 ++ 圈粉 ++ | 攻守兼备 |
| 爆料 + 爆料 | 连环爆料 | 赚钱 + 圈粉 +++ | 疯狂吸粉 |
| 爆料 + 谣言 | 惊天独家 | 圈粉 ++++ | 最强圈粉，不赚钱 |
| 谣言 + 谣言 | 阴谋论 | 圈粉 +++ | 高风险圈粉（见5.4） |
| 谣言 + 观点 | 舆论造势 | 圈粉 ++ 赚钱 + | |
| 观点 + 观点 | 行业综述 | 赚钱 ++ | 稳定但平庸 |
| 观点 + 爆料 | 独家评论 | 赚钱 + 圈粉 ++ | |

### 5.3 效果数值

文章强度 = 两张素材卡品质之和（2~6）

**赚钱效果（金币 = 文章强度 × 赚钱系数）：**

| 文章类型 | 赚钱系数 | 强度2→ | 强度4→ | 强度6→ |
|---------|---------|-------|-------|-------|
| 深度研报 | 3.0 | 6 | 12 | 18 |
| 专家点评 | 2.0 | 4 | 8 | 12 |
| 调查报道 | 1.5 | 3 | 6 | 9 |
| 行业综述 | 2.0 | 4 | 8 | 12 |
| 独家评论 | 1.0 | 2 | 4 | 6 |
| 舆论造势 | 1.0 | 2 | 4 | 6 |
| 连环爆料 | 0.5 | 1 | 2 | 3 |
| 惊天独家 | 0 | 0 | 0 | 0 |
| 阴谋论 | 0 | 0 | 0 | 0 |

**圈粉效果（影响力 = 文章强度 × 圈粉系数）：**

| 文章类型 | 圈粉系数 | 强度2→ | 强度4→ | 强度6→ |
|---------|---------|-------|-------|-------|
| 惊天独家 | 3.0 | 6 | 12 | 18 |
| 连环爆料 | 2.0 | 4 | 8 | 12 |
| 阴谋论 | 2.5 | 5 | 10 | 15 |
| 舆论造势 | 1.5 | 3 | 6 | 9 |
| 独家评论 | 1.5 | 3 | 6 | 9 |
| 调查报道 | 1.0 | 2 | 4 | 6 |
| 专家点评 | 0.5 | 1 | 2 | 3 |
| 深度研报 | 0 | 0 | 0 | 0 |
| 行业综述 | 0 | 0 | 0 | 0 |

### 5.4 阴谋论的风险机制

阴谋论（谣言+谣言）圈粉系数高，但有被打假的风险：

- 阴谋论发表后标记为"可被打假"
- 如果本回合被对手打假成功 → 圈粉效果归零，且**已有散户忠诚度 -1**
- 如果没被打假 → 正常生效

这给了打假文章一个特别有价值的目标。

## 6. 打假文章

### 6.1 打假的触发

打假是一种特殊的文章用途。玩家消耗素材卡合成打假文章，目标是**降低对手散户的忠诚度**。

消耗：1 行动点 + 1~2 张手牌

### 6.2 打假效果

打假影响力 = 使用的素材卡品质之和

效果：从对手的散户中选择目标，降低忠诚度：
- 每 1 点打假影响力 → 降低 1 点忠诚度
- 忠诚度降到 0 的散户 → 变为无主

### 6.3 素材类型对打假的加成

| 素材类型 | 打假加成 | 说明 |
|---------|---------|------|
| 数据卡 | ×1.5（向上取整） | 有理有据，打假效果最好 |
| 爆料卡 | ×1.0 | 正常效果 |
| 观点卡 | ×1.0 | 正常效果 |
| 谣言卡 | ×0.5（向下取整） | 以谣辟谣，效果很差 |

示例：用 1 张优质数据卡(品质2) 打假 → 影响力 = 2 × 1.5 = 3 → 降低 3 点忠诚度

### 6.4 打假 vs 阴谋论

如果对手本回合发了阴谋论，你可以选择：
- 正常打假（降低忠诚度）
- 针对阴谋论打假（取消圈粉效果 + 对手已有散户忠诚度 -1）

针对阴谋论打假更高效，给了玩家"留牌等对手出阴谋论"的策略空间。

---

## 7. 散户系统

### 7.1 散户属性

| 属性 | 说明 |
|------|------|
| 归属 | 无主 / 玩家A / 玩家B |
| 忠诚度 | 整数 ≥ 0 |

### 7.2 圈粉规则（和 v2 相同）

用文章的"圈粉影响力"争夺散户：

- **无主散户**：归属你，忠诚度 = 影响力值
- **自己的散户**：忠诚度 += 影响力值（巩固）
- **对手的散户**：先抵消忠诚度，归零后归你

圈粉时**优先影响无主散户**，无主散户用完后才影响对手散户。

### 7.3 忠诚度衰减

每回合结算阶段，所有有主散户忠诚度 **-1**。忠诚度降到 0 的散户变为无主。

这意味着：
- 你必须持续写圈粉文章来维护粉丝群
- 不维护的粉丝会自然流失
- 防止一方过早锁死所有散户

---

## 8. 影响力等级

玩家控制的散户数量决定影响力等级，影响力等级提供被动加成。

| 散户数量 | 影响力等级 | 每回合抽牌数 | 收益加成 |
|---------|-----------|------------|---------|
| 0~2 | Lv.1 | 3 张 | ×1.0 |
| 3~5 | Lv.2 | 4 张 | ×1.2 |
| 6~8 | Lv.3 | 5 张 | ×1.5 |
| 9~11 | Lv.4 | 6 张 | ×1.8 |
| 12+ | Lv.5 | 7 张 | ×2.0 |

**收益加成**：赚钱文章的金币收入 × 收益加成。圈粉效果不加成。

这样散户带来的优势是**赚钱更快**，而不是圈粉更快，避免赢家通吃的失控滚雪球。

## 9. 回合流程

```
┌─────────────────────────────────────────────────┐
│ 1. 抽牌阶段 DRAW                                 │
│    · 根据影响力等级抽取对应数量的素材卡             │
├─────────────────────────────────────────────────┤
│ 2. 行动阶段 ACTION                               │
│    · 玩家自由穿插以下操作，消耗行动点：             │
│      - 写赚钱文章：2张素材 + 1行动点 → 获得金币    │
│      - 写圈粉文章：2张素材 + 1行动点 → 抢夺散户    │
│      - 写打假文章：1~2张素材 + 1行动点 → 对手掉粉  │
│    · 行动点用完或玩家主动结束                       │
├─────────────────────────────────────────────────┤
│ 3. 结算阶段 SETTLEMENT                            │
│    · 所有有主散户忠诚度 -1                         │
│    · 忠诚度归零的散户变为无主                       │
│    · 检查游戏结束条件                              │
└─────────────────────────────────────────────────┘
```

### 9.1 行动点

| 参数 | 值 |
|------|-----|
| 每回合行动点 | 3 |
| 写赚钱文章 | 1 行动点 + 2 张素材卡 |
| 写圈粉文章 | 1 行动点 + 2 张素材卡 |
| 写打假文章 | 1 行动点 + 1~2 张素材卡 |

### 9.2 手牌

| 参数 | 值 |
|------|-----|
| 手牌上限 | 10 张 |
| 超出上限 | 回合结束时弃到上限 |

---

## 10. 胜负条件

8 回合结束时，**金币最多的玩家获胜**。

散户不折算为金币——散户是过程中的工具，不是最终目标。这避免了"最后一回合疯狂圈粉"的无意义操作。

---

## 11. 一回合完整示例

```
=== 第 4 回合 ===

玩家状态：金币 42 | 散户 5个（Lv.2）| 手牌 4张
AI状态：  金币 35 | 散户 4个 | 手牌 ?

[抽牌阶段]
  Lv.2 → 抽 4 张
  抽到：数据★★ + 谣言★ + 爆料★★ + 观点★
  当前手牌：4(旧) + 4(新) = 8 张

[行动阶段] 行动点 3/3

  思考：AI有4个散户在追我，我要不要打假？
  还是专心赚钱拉开差距？还是圈更多粉提升影响力？

  操作1：写赚钱文章（数据★★ + 观点★ = 专家点评，强度3）
  → 金币 = 3 × 2.0(赚钱系数) × 1.2(Lv.2加成) = 7.2 → 7金币
  → 当前金币：49
  行动点 2/3

  操作2：写圈粉文章（爆料★★ + 谣言★ = 惊天独家，强度3）
  → 圈粉影响力 = 3 × 3.0(圈粉系数) = 9
  → 抢 3 个无主散户（忠诚度 3, 3, 3）
  → 当前散户：8个 → Lv.3！
  行动点 1/3

  操作3：写打假文章（用剩余的 1 张数据★ 打假）
  → 打假影响力 = 1 × 1.5(数据加成) = 1（向上取整）
  → 对手 1 个散户忠诚度 -1
  行动点 0/3

[结算阶段]
  所有有主散户忠诚度 -1
  → 玩家：5个旧散户忠诚度各-1，3个新散户忠诚度各-1(3→2)
  → AI：4个散户忠诚度各-1，其中被打假的那个再-1（已在行动阶段扣过）
  → AI有1个散户忠诚度归零 → 变为无主

回合结束状态：
  玩家：金币 49 | 散户 8个（Lv.3）→ 下回合抽5张！
  AI：  金币 35 | 散户 3个 → 下回合抽3张……

  玩家这回合赚了钱、抢了粉、还打了假，三件事都做了。
  代价是手牌用了 5 张（2+2+1），剩 3 张留到下回合。
```

## 12. 待验证问题（playtest 清单）

- [ ] 2 张素材合成是否合适？会不会太少（组合太简单）或太多（手牌不够用）？
- [ ] 赚钱/圈粉系数是否平衡？前期圈粉 vs 赚钱的收益曲线交叉点在哪？
- [ ] 忠诚度每回合 -1 的衰减速度是否合适？太快→维护成本过高，太慢→先圈粉的人优势太大
- [ ] 打假 1~2 张卡的成本是否合适？太便宜→打假成为主要策略，太贵→没人打假
- [ ] 行动点 3 够不够？3 次行动 × 2 张卡 = 6 张卡，但抽牌最多 7 张，刚好?
- [ ] 阴谋论的风险/收益比是否合理？
- [ ] 影响力等级的阈值是否合适？Lv.2(3散户)是否太容易达到？
- [ ] 8 回合是否足够？游戏节奏（前期圈粉、中期赚钱、后期争夺）能否在 8 回合内展开？
- [ ] 数据加成打假 ×1.5 是否太强？会不会让数据卡变成万能卡？
- [ ] 初始 0 金币是否合适？第一回合只能圈粉不能赚钱？

## 13. 扩展方向

1. **声望系统** → 发阴谋论/谣言降声望，声望低了圈粉效率降低
2. **散户类型** → 保守型（喜欢数据文章）/激进型（喜欢爆料文章），不同文章对不同散户圈粉效率不同
3. **多人对抗** → 3~5 人争夺同一池散户
4. **写作方法卡** → 额外修饰卡，改变文章效果（如"标题党"：圈粉×2但赚钱×0.5）
5. **事件系统** → 每回合随机事件改变某类素材的价值（如"财报季"：数据卡赚钱系数翻倍）
6. **素材黑市** → 花金币购买指定类型素材卡
7. **连载文章** → 跨回合连续发表同类文章有 combo 加成
8. **粉丝画像** → 散户有偏好标签，匹配的文章圈粉效果翻倍
