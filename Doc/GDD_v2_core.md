# StockLegend 核心规则 v2

> 本文档定义单人最小可玩原型的完整规则。
> 不涉及内容设计（主题包装、文案、美术），只关注游戏流程和数值框架。
> 多人规则在单人验证通过后自然扩展。

## 2. 核心模型

### 2.1 两层结构：棋子与棋盘

```
散户 = 棋子（争夺对象）
股票 = 棋盘（散户的部署目标）

玩家行为：
  写文章 → 争夺散户归属权（涂地）
  指挥   → 派散户去买/卖某只股票（下棋）
  交易   → 玩家自己买卖股票（获利）
```

### 2.2 散户

每个散户是一个独立的棋子，拥有以下属性：

| 属性 | 说明 |
|------|------|
| 归属 | 无主 / 玩家A / 玩家B / ... |
| 忠诚度 | 整数 ≥ 0，对当前归属者的忠诚程度 |
| 部署状态 | 待命 / 已部署（买入某股票）/ 已部署（卖出某股票） |

```
散户状态示例：

  ●无主(0)  ●无主(0)  ●无主(0)    ← 无主散户，忠诚度0
  ★你(1)   ★你(2)   ★你(3)      ← 你的散户，忠诚度1/2/3
  ◆对手(1)  ◆对手(2)              ← 对手的散户

  ★你(2)→买入股票A               ← 已部署的散户
  ◆对手(1)→卖出股票A              ← 对手部署的散户
```

### 2.3 争夺散户（涂地）

玩家写文章影响散户，文章有一个**影响力**值。选择要影响的目标散户后：

**影响无主散户**：
- 散户归属变为你，忠诚度 = 影响力值

**巩固自己的散户**：
- 忠诚度 += 影响力值

**抢夺对手的散户**：
- 影响力先抵消对手的忠诚度
- 忠诚度降到 0 → 散户变为无主
- 若还有剩余影响力 → 散户归属你，忠诚度 = 剩余值

```
示例：你的影响力 = 3，目标是对手散户（忠诚度2）

  影响力 3 → 抵消忠诚度 2 → 散户变无主
           → 剩余 1 → 归属你，忠诚度 = 1

示例：你的影响力 = 2，目标是对手散户（忠诚度3）

  影响力 2 → 忠诚度从 3 降到 1 → 散户仍归对手
           （没能抢过来，但削弱了对方的控制）
```

### 2.4 指挥散户

玩家可以指挥**自己的待命散户**去执行买入或卖出某只股票的操作。

- 每次指挥消耗 1 行动点
- 指定：哪个散户 + 目标股票 + 买入或卖出
- 散户状态变为"已部署"

已部署的散户**持续产生买/卖压力**，直到被玩家撤回或被对手抢走。

**被抢走的已部署散户**：立即撤离部署位，回到新归属者的待命状态。

### 2.5 股价变动

每只股票的价格由**部署在该股票上的散户买卖力量差值**决定：

```
差值 = 该股票上的买方散户数 - 该股票上的卖方散户数
```

根据差值查阶梯表，在结算阶段更新股价。

**示例：标准型股票阶梯表**

| 差值区间 | 每回合涨跌 | 说明 |
|---------|-----------|------|
| ≤ -6 | -25% | 崩盘 |
| -4 ~ -5 | -15% | 大跌 |
| -2 ~ -3 | -8% | 下跌 |
| -1 | -3% | 微跌 |
| 0 | 0% | 横盘 |
| +1 | +3% | 微涨 |
| +2 ~ +3 | +8% | 上涨 |
| +4 ~ +5 | +15% | 大涨 |
| ≥ +6 | +25% | 暴涨 |

不同股票可以有不同的阶梯表来表达"性格"：
- **稳健型**：阶梯平缓，需要大量散户才有大波动
- **妖股型**：中间平坦两端陡峭，过了临界点就暴涨暴跌

## 3. 回合流程

每回合按以下阶段**顺序执行**：

```
┌─────────────────────────────────────────────────┐
│ 1. 世界阶段 WORLD                                │
│    · 预留：未来放系统事件（突发新闻、政策变动等）    │
│    · 最小原型中此阶段为空，直接进入取材             │
├─────────────────────────────────────────────────┤
│ 2. 取材阶段 GATHER                               │
│    · 从素材牌堆抽取 N 张素材卡加入手牌             │
│    · （扩展：声望越高，抽牌数/质量越好）            │
├─────────────────────────────────────────────────┤
│ 3. 行动阶段 ACTION                               │
│    · 玩家自由穿插以下操作，消耗行动点：             │
│      - 写文章：消耗素材卡 + 行动点 → 争夺散户       │
│      - 指挥散户：消耗行动点 → 派散户买/卖股票       │
│      - 自己交易：消耗行动点 + 资金 → 买/卖股票     │
│    · 行动点用完或玩家主动结束                       │
├─────────────────────────────────────────────────┤
│ 4. 结算阶段 SETTLEMENT                            │
│    · 无主散户随机行为：每个无主散户掷骰，产生买/卖  │
│      压力（见第7节）                               │
│    · 计算每只股票的总差值：                         │
│      玩家部署的散户差值 + 无主散户随机贡献           │
│    · 查阶梯表，按百分比更新股价                     │
│    · （扩展：忠诚度自然衰减机制）                   │
│    · 检查游戏结束条件                              │
└─────────────────────────────────────────────────┘
```

### 3.1 行动点分配

| 参数 | 值 | 备注 |
|------|-----|------|
| 每回合行动点 | 3 | playtest 后调整 |
| 写文章消耗 | 1 行动点 + 素材卡 | |
| 指挥散户消耗 | 1 行动点 | |
| 自己买入消耗 | 1 行动点 | |
| 自己卖出消耗 | 1 行动点 | |

玩家每回合自由组合：全部用来抢散户、或抢+部署+交易混搭、或全部用来交易……任意搭配。

## 4. 素材卡与文章

### 4.1 素材卡（最小原型简化版）

最小原型中，素材卡只有一个关键属性：**影响力**（1~3）。

| 素材卡等级 | 影响力 | 抽到概率 |
|-----------|-------|---------|
| 普通素材 | 1 | 50% |
| 优质素材 | 2 | 35% |
| 重磅素材 | 3 | 15% |

### 4.2 写文章

消耗 1 行动点 + 打出 1~3 张素材卡，选择目标散户：

- **影响力** = 所有素材卡影响力之和
- **目标** = 1 个散户（无主/自己的/对手的）
- 按 2.3 节规则执行争夺/巩固

示例：打出优质素材(2) + 普通素材(1) = 影响力 3
→ 选择一个对手的散户（忠诚度2），影响力3 - 忠诚度2 = 剩余1
→ 散户归属你，忠诚度 = 1

### 4.3 手牌

| 参数 | 值 |
|------|-----|
| 每回合抽牌数 | 3 张 |
| 手牌上限 | 7 张 |
| 超出上限 | 回合结束时弃到上限 |

## 5. 交易

### 5.1 玩家自己交易

- 买入：消耗 1 行动点，按当前股价买入 1 手（10 股）
- 卖出：消耗 1 行动点，按当前股价卖出 1 手（10 股）

### 5.2 散户 vs 玩家交易的区别

| | 散户部署 | 玩家自己交易 |
|---|---------|-------------|
| 作用 | 推动股价（制造买/卖压力） | 获取利润（低买高卖） |
| 花费 | 行动点（不花钱） | 行动点 + 资金 |
| 持续性 | 持续产生压力直到撤回 | 一次性成交 |

核心玩法：用散户操控股价，用自己的钱赚差价。

### 5.3 做空（最小原型暂不实现）

标记为扩展功能，验证核心循环后再加。

## 6. 胜负条件（单人模式）

6 回合结束时：
- 所有持仓按当前股价 × 50% 折算为现金（清算惩罚）
- **总资产 = 现金 + 清算所得**
- 按总资产评级：

| 总资产 | 评级 |
|--------|------|
| < 1000 | 亏损（F） |
| 1000~1200 | 保本（C） |
| 1200~1500 | 盈利（B） |
| 1500~2000 | 大赚（A） |
| > 2000 | 传奇（S） |

## 7. 无主散户的随机行为

无主散户每回合在结算阶段**自动产生随机的买卖压力**，直接叠加到股票的差值上。

**计算方式**：每个无主散户独立掷骰：

| 掷骰结果 | 效果 |
|---------|------|
| 1~2（33%） | 该散户本回合贡献 -1（卖出压力） |
| 3~4（33%） | 该散户本回合贡献 0（观望） |
| 5~6（33%） | 该散户本回合贡献 +1（买入压力） |

所有无主散户的贡献加总，叠加到股票的差值上。

```
示例：5 个无主散户，掷骰结果为 +1, 0, -1, +1, -1
→ 无主散户总贡献 = 0（互相抵消，市场平静）

示例：5 个无主散户，掷骰结果为 +1, +1, +1, 0, -1
→ 无主散户总贡献 = +2（散户集体看多，市场被随机推涨）
```

**设计意义**：
- 无主散户越多 → 每回合随机噪声越大 → 市场越不可预测
- 玩家争夺散户 → 无主散户减少 → 市场越可控
- 争夺散户的动机 = 获得兵力 + 消除噪声

## 8. 一回合完整示例

```
=== 第 3 回合 ===

当前状态：
  资金：800    持仓：1手（10股）@买入价90
  股价：95
  散户（12个）：
    你的散户(4个)：★(2)待命  ★(1)→买入A  ★(1)→买入A  ★(3)→卖出A
    无主散户(8个)：●●●●●●●●
    玩家部署差值：买方2 - 卖方1 = +1

[世界阶段]
  最小原型中跳过

[取材阶段]
  抽到：普通素材(1) + 优质素材(2) + 普通素材(1)
  当前手牌：5 张

[行动阶段] 行动点 3/3

  操作1：写文章（打出优质素材2 + 普通素材1 = 影响力 3）
  → 目标：1 个无主散户 → 归属你，忠诚度 = 3
  行动点 2/3

  操作2：指挥散户 → 把刚抢到的散户部署为"买入股票A"
  → 玩家部署差值：买方3 - 卖方1 = +2
  行动点 1/3

  操作3：结束行动（保留 1 点）

[结算阶段]
  无主散户(7个)掷骰：+1, -1, 0, +1, +1, -1, 0
  → 无主散户贡献 = +1

  总差值 = 玩家部署差值(+2) + 无主散户贡献(+1) = +3
  → 上涨档 → +8%
  股价：95 × 1.08 = 102.6 → 103

  复盘：自己部署了 +2 差值，无主散户碰巧贡献了 +1，合计 +3。
  运气不错。但如果 7 个无主散户骰出 -3，总差值就变成 -1 了……
  无主散户越多，这种翻盘的风险就越大。得加紧抢散户。
```

## 9. 待验证问题（playtest 清单）

- [ ] 12 个散户够不够？太少→没得抢，太多→管理复杂
- [ ] 忠诚度上限要不要封顶？无限堆忠诚度会不会导致"铁桶阵"无法攻破？
- [ ] 撤回已部署散户是否需要消耗行动点？还是免费？
- [ ] 指挥散户是"每个散户1行动点"还是"一次性指挥多个散户"？
- [ ] 一篇文章只能影响 1 个散户？还是可以分配给多个散户？
- [ ] 行动点 3 够不够？写文章(1) + 指挥(1) + 交易(1) 刚好用完，太紧？
- [ ] 散户部署"持续产生压力"直到撤回——这是否太强？需要自然衰减？
- [ ] 世界事件操控无主散户是否合理？是否需要更多样的事件类型？
- [ ] 阶梯表数值是否合理？12个散户最极端差值是±12
- [ ] 单人模式只有世界事件作为对手，张力够不够？是否需要AI对手？

## 10. 扩展方向（验证通过后逐步加入）

1. **多只股票** → 3~4 只，散户可以被部署到不同股票上，分兵策略
2. **声望系统** → 声望越高，抽牌越好/影响力加成；发假消息降声望
3. **打假机制** → 消耗素材卡攻击对手文章，降低对手散户忠诚度
4. **散户类型** → 保守型/激进型/跟风型，不同文章对不同类型影响力不同
5. **做空机制** → 允许玩家做空股票
6. **多人对抗** → 多玩家争夺同一池散户
7. **忠诚度衰减** → 每回合未巩固的散户忠诚度自然-1，需要持续经营
8. **写作方法卡** → 改变文章效果（群体影响、定向打击等）
9. **散户能力** → 某些散户有特殊能力（大户：部署时算2个买/卖压力）
