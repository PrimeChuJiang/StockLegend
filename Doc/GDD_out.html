StockLegend 玩法说明书

本文档面向美术团队，用通俗语言描述完整的游戏玩法和体验。 不涉及代码实现细节。

你在玩什么？

你扮演一个财经自媒体人，靠写文章影响股市舆论，再通过低买高卖赚取暴利。

你手里有两样东西：

  • 素材卡——各种财经信息碎片（数据、爆料、谣言、专家观点）

  • 写作方法卡——你掌握的写作技巧（标题党、春秋笔法、数据可视化等）

每天你把素材拼成文章，选个渠道发出去，操控市场情绪，然后趁机买卖股票。文章写得好就赚得多，写得烂可能被打假翻车。

1. 开盘

新的一天开始。如果今天是周一，公布本周日程表。如果今天日程上有事件，此时揭晓事件的具体内容（利好还是利空）。同时系统判定今天是否有突发新闻。所有事件的效果开始影响市场情绪。

2. 取材

你的准备时间。这个阶段可以做四件事（资源允许的话）：

  • 免费抽牌：从公共素材牌堆随机抽 3 张素材卡。抽到什么行业、什么类型全看运气

  • 定向取材（花人脉）：指定一个行业甚至具体素材类型，从牌堆中精准抽取。比如"我要科技行业的爆料卡"

  • 打探内幕（花人脉）：窥探日程上尚未到来的事件结果，提前知道是利好还是利空

  • 进修（花现金）：从随机出现的 3 张写作方法卡中学习 1 张新技巧，永久加入你的技能库

组合的搭配方式决定文章类型，不同类型有不同效果：

拼好的文章会显示两个关键属性：

  • 影响力——决定这篇文章能推动股价变动多快（数字越大，股价跑得越快）

  • 可信度——决定这篇文章的效果能持续多少天（数字越大，效果越持久），同时可信度越高，被打假的概率越低

撰写好的文章进入你的稿件区，等待发表。每天最多写 2 篇（受精力限制）。

4. 发表

决定稿件区的文章命运：

  • 发表——选一个渠道发出去，立刻开始影响市场情绪

  • 压稿——先不发，留到更好的时机。但文章有保鲜期（3 天），过期自动作废

  • 废稿——扔掉这篇文章，用掉的素材卡丢进弃牌堆

5. 市场反应

系统结算时间。根据当前所有生效中的影响（你的文章 + 环境事件），计算每只股票的情绪值，然后更新股价。

股价怎么变？ 情绪值控制的是股价变动的速度：

  • 情绪为正 → 股价往上涨，情绪越高涨得越快

  • 情绪为零 → 股价停在原地不动

  • 情绪为负 → 股价往下跌，情绪越低跌得越快

关键点：股价不会自动回弹。即使影响消退、情绪回到零，股价也只是停住不动，不会自己涨回去。想把股价拉回来，需要新的正面推力。

股价结算之后，系统对你今天发表的文章进行事实核查。可信度低的文章有较高概率被揭穿是假的：

  • 被打假后，文章的影响会反转（本来让股价跌的，反而开始推动股价涨）

  • 你的信誉下降

  • 使用的发表渠道被临时封禁一天

7. 收盘结算

一天结束。系统做几件事：

  • 计算你的总资产（现金 + 所有股票市值）

  • 所有文章和事件的影响"老化"一天，到期的自动消失

  • 压稿区的文章新鲜度减一，过期的自动作废

  • 检查是否达成胜利条件

四种素材卡

素材卡是你手里的"弹药"，代表你能获取到的各种财经信息。所有素材来自同一个公共牌堆，每天免费抽 3 张，也可以花人脉定向抽取。

每张素材卡有三个关键信息：

  • 行业标签——这张素材跟哪个行业相关（科技/金融/能源/医药/宏观）

  • 倾向——这条信息是利好、利空还是中性

  • 可信度和影响力——两个 1~5 的数值

方法卡分三种风格：

数值型——直接修改文章的影响力或可信度：

  • 「标题党」影响力大幅提升，但可信度下降

  • 「数据可视化」让数据卡的可信度更高

  • 「严谨论证」提高可信度但降低影响力

操控型——改变游戏规则：

  • 「春秋笔法」可以把一条利好消息写成利空的（或反过来）

  • 「断章取义」可以把普通数据伪装成爆料

  • 「匿名化处理」降低被打假的概率

特殊型——提供独特能力：

  • 「病毒传播」让文章同时影响整个行业的所有股票

  • 「连续剧」如果连续两天对同一只股票发文，影响力翻倍

  • 「深喉网络」本轮定向取材免费

方法卡的稀有度分三级：普通（进修时经常出现）、稀有（偶尔出现）、罕见（很少出现，效果最强）。

退市

如果某只股票被砸到初始价的 20% 以下（比如星海科技跌破 10 元），就会触发退市：

  • 这只股票从游戏中彻底消失

  • 你手里如果持有这只股票，会被按退市价强制卖出（大概率巨亏）

  • 之后不能再对这只股票写文章或交易

退市是一把双刃剑——如果你是故意砸盘的，说明你已经赚完了；如果你没来得及跑，那就是灾难。

被打假的后果很严重：

  • 文章的效果反转——本来推股价跌的，反而开始推股价涨（或者反过来）

  • 信誉大幅下降（-10 分）

  • 使用的发表渠道被临时封禁一天

但有个微妙之处：打假发生在股价结算之后。也就是说，你发的假文章当天确实影响了股价，你当天确实可以按已经变动的价格交易——打假的惩罚要到明天才生效。这就给了你一个"打一枪就跑"的窗口，但如果贪心没跑掉，明天的反转会让你亏得更惨。

日程事件 vs 突发事件

  • 日程事件：每周一公布，你知道哪天会发生什么（但不知道好坏）。可以花人脉提前打探

  • 突发事件：每天 30% 概率随机冒出来，完全不可预料也无法提前打探

环境与你的文章互动

  • 顺势：环境利空 + 你也写利空 → 你的文章影响力加成（顺应民意，大家更容易相信）

  • 逆势：环境利好 + 你写利空 → 你的文章可信度下降（大家觉得你在胡说）
