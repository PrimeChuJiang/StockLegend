# StockLegend 游戏策划案 v1

> 第一版策划案，基于初步设计讨论整理。
> 本文档定义核心玩法循环、系统规则和数据结构，作为后续开发的基准。

---

## 1. 游戏概述

### 1.1 一句话描述

你是一个无良财经自媒体人，通过撰写真假参半的文章操控舆论、影响股价，从中低买高卖赚取暴利。

### 1.2 核心体验

- **身份幻想**：幕后的市场操控者，用笔杆子撬动资本市场
- **策略深度**：素材选择 → 文章组合 → 发表时机 → 交易决策，环环相扣
- **黑色幽默**：荒诞的市场事件、自我打脸的分析师、"编的比真的还准"的假新闻

### 1.3 核心循环

```
取材（抽牌）→ 撰写（组合素材卡）→ 发表（触发舆论效果）→ 交易（买卖股票获利）
```

### 1.4 胜利条件

在限定回合内（暂定 20 回合），资产总额（现金 + 持仓市值）达到目标值。

## 2. 回合结构

### 2.1 五日周期制

游戏以 **5 个回合为一个周期**（模拟一个交易周），总计 4 个周期共 20 回合。

每个周期第 1 天（第 1、6、11、16 回合），系统公布**本周期 5 天的新闻日程表**。日程表显示哪些天会发生什么类型的事件，但**不透露事件结果是利好还是利空**。玩家利用这个规划窗口提前布局：备素材、写稿件、建仓位。

```
┌─ 第二周期日程公布（第6回合开始时）─────────────────────┐
│                                                        │
│  第 6天：（无预定事件）                                 │
│  第 7天：星海科技 · 季度财报发布                         │
│  第 8天：（无预定事件）                                 │
│  第 9天：央行 · 议息会议结果公布                         │
│  第10天：仁和医药 · 新药三期临床结果                     │
│                                                        │
│  ※ 事件结果未知，可消耗人脉提前窥探                      │
└────────────────────────────────────────────────────────┘
```

**关键时间线**：玩家在第 6 天看到日程后，有第 6~7 天的时间为星海科技财报做准备；有第 6~9 天为央行议息做准备。越早的事件留给玩家的准备窗口越短。

### 2.2 内幕窥探机制

玩家可以消耗**人脉**提前查看尚未揭示的日程事件结果：

```
窥探规则：
  · 目标：本周期内尚未到来的日程事件
  · 成本：1 人脉/次
  · 效果：立即揭示该事件的完整内容（利好/利空、影响力大小）
  · 时机：GATHER 阶段可执行窥探（与取材同阶段）
  · 限制：每回合最多窥探 1 次
```

窥探的价值与时机挂钩——越早窥探，可操作的回合越多，但人脉是有限资源，和定向取材、主流媒体发稿竞争：

```
人脉的三种用途（互相竞争）：
  · 窥探日程事件 → 信息优势（提前知道好坏，精准布局）
  · 定向取材     → 素材优势（获取指定行业的素材卡）
  · 主流媒体发稿 → 影响力优势（更大的文章效果）
```

这本质上就是**内幕交易**——花人脉"找关系打听消息"，然后利用信息差获利。完美契合游戏的黑色幽默主题。

### 2.3 单日回合阶段

每回合（每个交易日）按以下阶段顺序执行：

```
┌─────────────────────────────────────────────────────────┐
│ 1. 开盘阶段 MARKET_OPEN                                 │
│    · 若为周期第1天（5n+1），公布本周期5日日程表            │
│    · 揭示本回合的日程事件（如有），其结果此时自然公开        │
│    · 判定是否触发突发新闻（30%概率，从突发事件牌堆抽取）    │
│    · 所有环境牌效果生效，修改各股票情绪值                   │
├─────────────────────────────────────────────────────────┤
│ 2. 取材阶段 GATHER                                      │
│    · 从素材牌堆随机抽取 N 张（默认 3 张）                 │
│    · 可消耗人脉进行定向取材（见 3.3 节）                   │
│    · 可消耗人脉窥探未来日程事件（见 2.2 节）               │
│    · 可消耗现金进修，获取新的写作方法卡（见 4.2 节）        │
│    · 当前回合已揭示的环境事件加入"可引用素材池"            │
├─────────────────────────────────────────────────────────┤
│ 3. 撰写阶段 WRITING                                     │
│    · 从手牌中选择 2~4 张素材卡组合成文章                   │
│    · 系统实时计算文章属性（类型/冲击力/可信度/风险）         │
│    · 可撰写多篇文章，受精力值限制                          │
│    · 文章完成后进入"稿件区"，等待发表或继续压稿             │
├─────────────────────────────────────────────────────────┤
│ 4. 发表阶段 PUBLISH                                     │
│    · 决定稿件区的文章：发表 / 继续压稿 / 废稿              │
│    · 选择发表渠道（自媒体/主流媒体/匿名论坛/付费推广）     │
│    · 发表的文章效果生效，修改对应股票的情绪值               │
├─────────────────────────────────────────────────────────┤
│ 4.5 澄清阶段 CLARIFY（发表后立即触发）                    │
│    · 每篇文章发表后，其他玩家进入"澄清窗口"               │
│    · 玩家选择是否对该文章发起打假                          │
│    · 选择打假的玩家合成反驳文章，进行影响力比对             │
│    · 详见第 16 节：文章打假系统                            │
├─────────────────────────────────────────────────────────┤
│ 5. 市场反应阶段 MARKET_REACT                             │
│    · 结算所有股票的最终情绪值                              │
│    · 对已发表文章执行"事实核查"判定                        │
│    · 根据情绪值计算各股票本回合涨跌幅，更新股价             │
├─────────────────────────────────────────────────────────┤
│ 6. 交易阶段 TRADE                                       │
│    · 玩家买入/卖出股票（按更新后的股价结算）                │
│    · 每回合交易次数有限（暂定 3 次）                       │
├─────────────────────────────────────────────────────────┤
│ 7. 收盘结算阶段 SETTLEMENT                               │
│    · 计算玩家当前资产总额（现金 + 持仓市值）                │
│    · 所有股票情绪值向 0 衰减（衰减率暂定 30%）             │
│    · 压稿区文章"新鲜度"衰减（过期文章影响力下降）           │
│    · 检查胜负条件                                        │
└─────────────────────────────────────────────────────────┘
```

### 2.4 阶段枚举定义

```
Phase {
    MARKET_OPEN,    # 开盘
    GATHER,         # 取材
    WRITING,        # 撰写
    PUBLISH,        # 发表
    CLARIFY,        # 澄清（打假窗口）
    MARKET_REACT,   # 市场反应
    TRADE,          # 交易
    SETTLEMENT      # 收盘结算
}
```

## 3. 素材卡系统

### 3.1 素材卡类型

素材卡是玩家的核心手牌，代表可用于撰写文章的信息碎片。每张素材卡有三种用途，用在一处就不能用在另一处：

```
素材卡的三种用途（互斥消耗）：
  · 写文章 → 组合素材发表，影响股价（见第 5 节）
  · 卖到黑市 → 换取现金（见第 15 节：情报黑市）
  · 作为打假弹药 → 合成反驳文章攻击对手（见第 16 节：文章打假）
```

这个设计的核心在于：同一张卡在不同用途之间存在取舍。一张高可信度数据卡，你是拿去写文章操控股价？卖掉换钱炒股？还是留着等对手发假新闻时合成辟谣文章锤他？

| 类型 | 说明 | 可信度倾向 | 影响力倾向 |
|------|------|-----------|-----------|
| **数据卡 DATA** | 财报摘要、行业统计、K线图表 | 高（3~5） | 低（1~3） |
| **爆料卡 SCOOP** | 内幕消息、泄露文件、知情人爆料 | 中（2~4） | 高（3~5） |
| **谣言卡 RUMOR** | 小道消息、论坛传闻、"据知情人士透露" | 低（1~2） | 极高（4~5） |
| **观点卡 OPINION** | 专家评论、机构评级、分析师观点 | 中（2~4） | 中（2~4） |

素材卡只有这四种类型，代表"写什么"。"怎么写"由写作方法卡决定（见第 4 节）。

### 3.2 素材卡属性

每张素材卡拥有以下属性：

```
MaterialCard {
    id: StringName          # 唯一标识
    name: String            # 素材名称，如"星海科技Q3财报"
    type: MaterialType      # DATA / SCOOP / RUMOR / OPINION
    industry: Industry      # 所属行业标签（可为 MACRO 表示宏观通用）
    bias: Bias              # 倾向：BULLISH（利好）/ BEARISH（利空）/ NEUTRAL（中性）
    credibility: int        # 可信度 1~5
    impact: int             # 影响力 1~5
    tags: Array[StringName] # 语义标签，用于协同/冲突判定（如 "财务造假"、"高管变动"）
    flavor_text: String     # 风味文字（黑色幽默描述）
}
```

### 3.3 取材机制：统一牌堆 + 定向抽取

所有素材卡属于**同一个牌堆**，不按类型或行业分开存放。抽取方式分为两种：

**普通取材（免费）**

每回合 GATHER 阶段自动从牌堆顶部随机抽取 N 张（暂定 3 张），不筛选行业或类型。可能抽到任何行业、任何类型的素材。

- 优点：免费，每回合都有
- 缺点：随机性高，可能抽到与当前计划无关的行业素材

**定向取材（消耗人脉）**

玩家消耗人脉从牌堆中筛选指定条件的素材抽取，精确度分两级：

| 定向级别 | 人脉成本 | 效果 |
|---------|---------|------|
| 行业定向 | 1 人脉 | 指定一个行业，从牌堆中筛选出该行业的素材，随机抽 1 张 |
| 精确定向 | 2 人脉 | 指定行业 + 素材类型（如"科技行业的爆料卡"），从牌堆中筛选后随机抽 1 张 |

- 定向取材在普通取材之后执行，每回合可执行多次（受人脉限制）
- 若牌堆中没有符合条件的素材，人脉不消耗，取材失败

设计意图：普通取材提供基础资源流，定向取材让玩家能为特定计划精准备料，但与窥探、主流媒体发稿竞争同一资源（人脉）。"我是省着人脉打听内幕，还是花掉去找我需要的素材？"——这是每回合的核心取舍。

## 4. 写作方法卡系统

### 4.1 概述

写作方法卡代表玩家掌握的**写作技巧和手法**——不是你写什么，而是你怎么写。

```
两类卡牌的本质区别：
  素材卡 = 你写什么（信息、证据、谣言）→ 公共牌堆，抽取获得，用完消耗
  写作方法卡 = 你怎么写（技巧、角度、风格）→ 个人卡库，进修获得，永久持有
```

### 4.2 获取方式：角色自带

~~进修系统已移除。~~ 写作方法卡通过**角色选择**获得——每个角色自带 4~5 张专属写作方法卡，这些卡从游戏开始到结束固定不变。不同角色拥有不同的写作方法组合，决定了玩家的文章风格和策略路线。

```
获取规则：
  · 角色选定时，自动获得该角色的全部专属写作方法卡
  · 局内无法获取新的写作方法卡
  · 不同角色之间的写作方法卡可能有重叠（如多个角色都有「严谨论证」）
  · 所有角色额外持有 1 张通用基础卡「平铺直叙」（保底方法）
```

设计意图：角色选择即是"构建"——你选了谁，就决定了你的写作工具箱。这取代了进修系统的渐进式构建，让玩家从第一回合开始就有完整的能力，降低上手门槛。同时，角色差异化激励玩家尝试不同角色进行多盘游戏。详见第 17 节：角色系统。

### 4.3 通用基础卡

所有角色共享 **1 张通用写作方法卡**：

| 初始卡牌 | 效果 |
|---------|------|
| 「平铺直叙」 | 无特殊效果，文章按基础公式计算（保底方法，确保永远能写文章） |

### 4.4 使用规则

```
· 每篇文章可附加 0~2 张写作方法卡
· 写作方法卡不消耗，使用后留在个人卡库
· 每张卡每回合最多使用 1 次
· 使用过的卡进入冷却，下回合不可使用
· 冷却持续 1 回合，之后自动恢复可用
```

冷却机制的效果：如果卡库中有 6 张方法卡，每回合最多可用其中一半左右（另一半在冷却），这鼓励玩家积累多样化的卡库而非只靠一两张强力卡。

### 4.5 写作方法卡数据结构

```
WritingMethodCard {
    id: StringName          # 唯一标识
    name: String            # 如"标题党"、"交叉验证"
    description: String     # 效果描述
    effect: Dictionary      # 效果参数（见下方效果类型）
    rarity: Rarity          # COMMON / UNCOMMON / RARE
    flavor_text: String     # 黑色幽默描述
    is_on_cooldown: bool    # 运行时状态：是否在冷却中
}
```

### 4.6 写作方法卡一览

**基础类（影响数值）**：

| 卡牌名 | 稀有度 | 效果 |
|-------|-------|------|
| 标题党 | 普通 | 影响力 ×1.5，可信度 ×0.7 |
| 数据可视化 | 普通 | 文章中每张数据卡可信度 +1 |
| 情感渲染 | 普通 | 影响力 +3，事实核查概率 +15% |
| 严谨论证 | 稀有 | 可信度 +2，影响力 -1 |
| 交叉验证 | 罕见 | 文章中每有两张不同类型素材，可信度 +1 |

**操控类（改变规则）**：

| 卡牌名 | 稀有度 | 效果 |
|-------|-------|------|
| 春秋笔法 | 稀有 | 可将 1 张素材的 bias 反转（利好↔利空） |
| 匿名化处理 | 普通 | 事实核查概率减半，可信度 -1 |
| 断章取义 | 稀有 | 可将数据卡或观点卡当作爆料卡使用（改变类型参与组合判定） |
| 长期追踪 | 罕见 | 文章情绪修正器持续时间 +2 回合 |

**特殊类（独特能力）**：

| 卡牌名 | 稀有度 | 效果 |
|-------|-------|------|
| 病毒传播 | 罕见 | 文章同时影响目标行业的所有股票 |
| 连续剧 | 稀有 | 若上回合对同一股票发过文章，本篇影响力 ×2 |
| 危机公关 | 罕见 | 本篇文章不产生新修正器，改为移除目标股票上一个己方修正器并反转 |
| 深喉网络 | 稀有 | 本回合定向取材不消耗人脉 |

稀有度影响在进修时出现的概率：

| 稀有度 | 出现权重 |
|-------|---------|
| 普通 COMMON | 60% |
| 稀有 UNCOMMON | 30% |
| 罕见 RARE | 10% |

## 5. 文章组合系统

### 5.1 组合规则

```
一篇文章 = 2~4 张素材卡 + 0~2 张写作方法卡
```

- 素材卡必须**行业标签一致**，或使用 MACRO（宏观）标签的万能素材
- 写作方法卡无行业限制，任何文章都能用
- 每回合撰写文章数量受**精力值**限制（暂定每篇消耗 1 精力，每回合 2 精力）

### 5.2 文章属性计算

组合完成后，系统分三步计算文章属性：

**第一步：基础值（由素材决定）**

```
基础可信度 = floor(所有素材卡可信度的平均值)
基础影响力 = 所有素材卡影响力之和
文章方向   = 统计素材倾向多数决（利好多→利好，利空多→利空，平票→中性）
目标行业   = 素材共有的行业标签
```

**第二步：文章类型加成（由素材组合模式决定）**

| 组合模式 | 文章类型 | 特殊效果 |
|---------|---------|---------|
| 数据 + 数据 + 观点 | 深度研报 | 持续时间 +1 回合 |
| 爆料 + 数据 | 调查报道 | 可信度 +1 |
| 谣言 + 爆料 | 惊天独家 | 影响力 ×1.5，事实核查概率翻倍 |
| 谣言 + 谣言 | 阴谋论 | 50%概率影响力翻倍，50%概率文章无效 |
| 爆料 + 爆料 | 连环爆料 | 影响力 +3，事实核查概率 +20% |
| 数据 + 观点 | 专家点评 | 可信度 +1 |
| 观点 + 观点 | 舆论造势 | 影响力 +2 |
| 其他组合 | 一般报道 | 无特殊效果 |

**第三步：写作方法卡修正**

依次应用附加的写作方法卡效果（加法先于乘法）。

```
最终可信度 = floor((基础可信度 + 类型加成 + 方法卡加成) × 方法卡乘数)
最终影响力 = floor((基础影响力 + 类型加成 + 方法卡加成) × 方法卡乘数)
```

示例：2 张数据卡（可信度 4,3 影响力 2,2）+ 1 张观点卡（可信度 3 影响力 3）+ 写作方法卡「标题党」

```
第一步：基础可信度 = floor((4+3+3)/3) = 3，基础影响力 = 2+2+3 = 7
第二步：匹配"数据+数据+观点=深度研报"，持续时间+1（数值不变）
第三步：「标题党」影响力 ×1.5，可信度 ×0.7
最终：可信度 = floor(3 × 0.7) = 2，影响力 = floor(7 × 1.5) = 10
      持续时间 = 可信度2 + 类型加成1 = 3 回合
```

### 5.3 素材间的化学反应

同类型文章之间的差异不仅来自数值高低，还来自素材间的协同与冲突关系：

**协同（Synergy）**——满足条件时提供额外加成：

| 条件 | 效果 | 设计意图 |
|------|------|---------|
| 两张素材 tags 有交集且来自不同 type | 可信度 +1（证据链） | 多源交叉验证更可信 |
| 爆料卡 + 数据卡指向同一结论（bias 一致） | 影响力 +2 | 有理有据的爆料更震撼 |
| 引用当前回合环境事件作为素材 | 可信度 +1（蹭热点） | 真实事件背书 |

**冲突（Contradiction）**——不良搭配导致惩罚：

| 条件 | 效果 | 设计意图 |
|------|------|---------|
| 同一篇文章中 bias 互相矛盾（既利好又利空） | 影响力 ×0.5（方向混乱） | 自相矛盾的文章没人信 |
| 多张谣言卡堆叠（≥2） | 可信度额外 -1 每张（越编越离谱） | 防止无脑堆谣言 |
| 高可信度素材（≥4）与低可信度素材（≤2）混用 | 整体可信度被低者拖累，按低者计算 | 一颗老鼠屎坏一锅粥 |

## 7. 市场系统

### 7.1 股票

游戏中包含若干只股票，分属不同行业。初版暂定：

| 股票名称 | 行业 | 初始股价 | 波动性 |
|---------|------|---------|-------|
| 星海科技 | 科技 TECH | 50 | 高 |
| 中原银行 | 金融 FINANCE | 80 | 低 |
| 绿源能源 | 能源 ENERGY | 40 | 中 |
| 仁和医药 | 医药 MEDICAL | 60 | 中 |

**波动系数**决定每点情绪值推动股价变动的绝对金额：

| 波动性 | 波动系数 | 含义 |
|--------|---------|------|
| 低 LOW | 0.5 | 每点情绪每回合推动股价 ±0.5 元 |
| 中 MEDIUM | 1.0 | 每点情绪每回合推动股价 ±1.0 元 |
| 高 HIGH | 1.5 | 每点情绪每回合推动股价 ±1.5 元 |

### 7.2 情绪值系统（修正器驱动）

每只股票的市场情绪值由**所有活跃的情绪修正器之和**决定，范围夹到 -10 ~ +10：

```
-10 ─────── -5 ─────── 0 ─────── +5 ─────── +10
极度恐慌    偏空       中性       偏多      极度乐观
```

```
当前情绪值 = clamp(Σ 所有活跃修正器的 value, -10, +10)
```

情绪修正器来自两个来源，格式统一：

```
SentimentModifier {
    source_id: StringName   # 来源标识（文章ID 或 环境牌ID）
    source_type: String     # "article" 或 "environment"
    target_stock: StringName
    value: int              # 修正值（正=利好，负=利空）
    remaining_turns: int    # 剩余持续回合数
}
```

**文章产生的修正器**：
- value = 文章影响力 × 方向（利好 +1 / 利空 -1）
- remaining_turns = 文章可信度（影响力决定幅度，可信度决定持续时间）

**环境牌产生的修正器**：
- value = 环境牌的 sentiment_modifier
- remaining_turns = 环境牌的 duration

**修正器生命周期**：
- MARKET_OPEN 阶段：环境牌修正器施加
- PUBLISH 阶段：文章修正器施加
- SETTLEMENT 阶段：所有修正器 remaining_turns -= 1，归零的修正器移除

没有任何活跃修正器时，情绪值为 0，股价不再变动（定格在当前位置）。

### 7.3 情绪值 → 股价变动（速度模型）

**情绪值控制股价变动的速度，而非股价本身。** 每回合 MARKET_REACT 阶段：

```
每回合股价变动 = 情绪值 × 波动系数
新股价 = 当前股价 + 情绪值 × 波动系数
```

核心特性：
- 情绪 > 0 → 股价持续上涨（涨速与情绪值成正比）
- 情绪 = 0 → 股价**定格在当前位置**，不会回弹到初始价
- 情绪 < 0 → 股价持续下跌

股价是**路径依赖**的——历史上所有情绪修正器的累积效果决定了当前价格。修正器到期只意味着"不再继续推动"，不意味着"之前的推动被撤销"。

以星海科技（初始价 50，高波动，系数 1.5）为例：

| 情绪值 | 每回合变动 | 含义 |
|--------|-----------|------|
| +10 | +15.0 元/回合 | 极度乐观，股价快速上涨 |
| +5 | +7.5 元/回合 | 偏多，股价稳步上涨 |
| 0 | 0 元/回合 | 中性，股价不动 |
| -5 | -7.5 元/回合 | 偏空，股价稳步下跌 |
| -10 | -15.0 元/回合 | 极度恐慌，股价快速下跌 |

### 7.4 股价与情绪的时序关系

**股价每回合只结算一次**，在 MARKET_REACT 阶段计算。回合内的时序：

```
MARKET_OPEN  → 环境牌修正器生效（情绪值变化）
GATHER       → 取材（情绪值不变）
WRITING      → 撰写（情绪值不变）
PUBLISH      → 文章修正器生效（情绪值变化）
MARKET_REACT → 根据当前情绪值计算股价变动 → 然后执行事实核查
TRADE        → 玩家按新股价买卖
SETTLEMENT   → 修正器 tick，到期的移除
```

关键设计：**MARKET_REACT 阶段先结算股价，再执行事实核查**。即使文章当回合就被打假，当回合的股价变动已经发生。打假的惩罚体现在后续回合（见 8.1 节）。

玩家的典型操作流：

```
前几天 TRADE  → 提前建仓（买入目标股票）
当天 PUBLISH  → 发表利好文章，情绪修正器生效
当天 MARKET_REACT → 股价上涨
当天 TRADE    → 卖出获利
```

**操作顺序极其重要**——必须先建仓再发文，否则你自己把价格推高了再去买就亏了。

### 7.5 退市机制

当某只股票价格跌破**初始价的 20%** 时，触发退市：

| 股票 | 初始价 | 退市线 |
|------|-------|-------|
| 星海科技 | 50 | 10 |
| 中原银行 | 80 | 16 |
| 绿源能源 | 40 | 8 |
| 仁和医药 | 60 | 12 |

**退市流程**：

```
MARKET_REACT 阶段，股价结算后检查退市条件：
  1. 该股票标记为"已退市"
  2. 玩家该股票的所有持仓按退市价强制清算（通常亏损惨重）
  3. 该股票从游戏中移除：
     · 不再参与情绪/股价结算
     · 不再触发相关环境事件（日程中的未来事件取消）
     · 针对该股票行业的素材卡仍留在牌堆中，但若该行业已无上市股票则变为死牌
  4. 触发特殊事件文本："恭喜你，你用几篇文章把一家上市公司写没了"
```

退市是**不可逆的**——消失了就消失了。这对玩家是双刃剑：
- 如果你做空赚到了钱再退市，等于"完美犯罪"——但你也失去了一个未来可操控的标的
- 如果你持仓被强制清算，那就是翻车现场

### 7.6 股价变动完整示例

```
星海科技，初始价 50，波动系数 1.5，退市线 10

第6回合：
  MARKET_OPEN → 环境牌「芯片禁令」修正器 {value: -3, remaining: 2}
  PUBLISH     → 玩家利空文章修正器     {value: -8, remaining: 1}
  情绪值 = clamp(-3 + -8, -10, 10) = -10
  MARKET_REACT → 股价 = 50 + (-10 × 1.5) = 35
  SETTLEMENT  → 两个修正器 remaining 各 -1

第7回合：
  利空文章修正器到期移除
  芯片禁令修正器还剩 1 回合，value = -3
  情绪值 = -3
  MARKET_REACT → 股价 = 35 + (-3 × 1.5) = 30.5
  SETTLEMENT → 芯片禁令修正器 remaining -1

第8回合：
  芯片禁令修正器到期移除
  无活跃修正器，情绪值 = 0
  MARKET_REACT → 股价 = 30.5 + 0 = 30.5（定格，不会回到50）

第9回合：
  玩家发表利好文章修正器 {value: +5, remaining: 3}
  情绪值 = +5
  MARKET_REACT → 股价 = 30.5 + (5 × 1.5) = 38.0（开始回升，但靠的是新的推力）
```

设计意图：股价是所有历史情绪的累积结果。"推下去容易拉回来难"——砸盘只需要一波利空，但修复股价需要持续不断的利好推力。这迫使玩家认真规划操控节奏，而不是随意乱砸。

## 8. 环境牌系统

### 8.1 环境牌层级

| 层级 | 影响范围 | 示例 |
|------|---------|------|
| **宏观事件 MACRO** | 所有股票 | 央行降息、全球衰退、贸易战 |
| **行业事件 INDUSTRY** | 某行业所有股票 | 芯片禁令、医药集采、新能源补贴 |
| **个股事件 COMPANY** | 单只股票 | CEO丑闻、财报暴雷、产品召回 |

### 8.2 抽取机制：五日周期日程 + 突发新闻

**新闻日程表（可预见，五日周期公布）**

每个周期第 1 天（第 1、6、11、16 回合），系统从日程事件牌堆中抽取若干张，分配到本周期的 5 天中，公布日程表。

```
日程生成规则（暂定）：
  · 每个周期 2~3 个日程事件，随机分配到 5 天中
  · 日程事件只展示"事件标题"（如"星海科技 财报发布"），不展示结果
  · 事件结果（利好/利空/影响力大小）在事件当天 MARKET_OPEN 阶段揭示
  · 玩家可在事件日之前消耗人脉窥探结果（见 2.2 节）
```

信息三层级：

```
免费信息（日程公布时）：知道"第7天星海科技有财报事件"
付费窥探（事件日前）：  花1人脉提前得知"财报暴雷，利空-4"
自然揭示（事件当天）：  所有人看到完整结果
突发新闻（完全未知）：  不在日程上，无法窥探
```

设计意图：日程表给玩家一个规划窗口，玩家围绕已知日程提前备稿建仓。窥探机制模拟内幕交易，是花人脉获取信息差还是省人脉做其他事——这是核心资源决策。

**突发新闻（不可预见）**

每回合 MARKET_OPEN 阶段额外判定：

```
突发新闻概率：30%（暂定）
触发时从"突发事件牌堆"随机抽取一张，立即生效
突发事件不在日程表上显示，无法被窥探
```

突发事件制造计划外的混乱，但也创造机会——市场越乱，舆论操控的空间越大。

### 8.3 环境牌数据结构

```
EnvironmentCard {
    id: StringName
    name: String                   # 如"央行宣布降息50个基点"
    preview_name: String           # 日程表上显示的预告名（如"央行 · 议息会议结果公布"）
    tier: EventTier                # MACRO / INDUSTRY / COMPANY
    target_industry: Industry      # tier=INDUSTRY时指定行业；MACRO时为ALL
    target_stock: StringName       # tier=COMPANY时指定个股；否则为空
    sentiment_modifier: int        # 对目标情绪值的修正（如 +3 或 -4）
    duration: int                  # 持续回合数（1=仅本回合，>1=持续影响）
    source_type: EventSourceType   # SCHEDULED（日程）或 BREAKING（突发）
    can_be_referenced: bool        # 玩家是否可引用为写作素材
    is_revealed: bool              # 运行时状态：结果是否已被揭示（窥探或自然揭示）
    flavor_text: String
}
```

### 8.4 环境牌与文章的交互

**顺势/逆势修正**：

- 玩家文章方向与当前环境一致（顺势）→ 文章影响力 ×1.3
- 玩家文章方向与当前环境矛盾（逆势）→ 文章可信度 ×0.7
- 设计意图：顺势省力但利润空间小（趋势已被环境消化），逆势费力但成功后利润巨大（独家反向布局）

**引用环境事件为素材**：

- 当前回合已揭示的环境事件可被引用为写作素材
- 引用的环境事件视为一张**临时数据卡**（可信度 4，影响力 2，bias 与事件一致）
- 引用不消耗环境牌本身（多篇文章可引用同一事件）

## 9. 风险与信誉系统

### 9.1 事实核查

**判定时机**：文章发表当回合的 MARKET_REACT 阶段，**在股价结算之后**执行一次性判定。之后不再复查——信息过了时效就没有打假的价值了。

```
MARKET_REACT 阶段内部时序：
  1. 根据当前情绪值计算股价变动（文章效果已生效）
  2. 对本回合发表的每篇文章执行事实核查判定
```

**核查概率**：

```
核查触发概率 = 基础概率(10%) + 谣言卡数量 × 15% + 可信度惩罚
    其中可信度惩罚：可信度≤2时 +20%，可信度≤1时 +40%
    上限：90%
```

| 文章可信度 | 大致打假概率 | 风险画像 |
|-----------|------------|---------|
| 5 | ~10% | 几乎不翻车，效果持续 5 回合 |
| 3 | ~20% | 偶尔翻车，效果持续 3 回合 |
| 1 | ~50% | 约一半概率翻车，效果仅 1 回合 |

可信度同时降低被打假风险并延长效果持续时间——**可信度是纯粹的"稳定性"属性**，与影响力的"爆发力"形成干净对立。

**被打假后果**：

```
打假处理流程：
  1. 原文章的情绪修正器立即移除
  2. 施加反转修正器：value = 原修正值反向 × 0.5，remaining = 2 回合
  3. 玩家信誉值 -10
  4. 使用的发表渠道进入冷却（1 回合不可用）
```

**关键设计**：当回合股价已经按原修正器结算过了，打假不会撤销当回合的股价变动。惩罚体现在后续回合的反转。这产生了赌徒式决策：

```
示例——发表一篇低可信度利空文章：

第6回合：
  PUBLISH      → 利空文章，修正器 {value: -8, remaining: 1}
  MARKET_REACT → 股价暴跌（-8 生效）→ 事实核查：被打假！
                 → 原修正器移除
                 → 施加反转修正器 {value: +4, remaining: 2}
                 → 信誉 -10
  TRADE        → 此时股价仍是暴跌后的价格（可以按低价买入）

第7回合：
  MARKET_REACT → 反转修正器 +4 → 股价反弹
  （如果你第6回合贪心买入了，现在就被套了）

第8回合：
  反转修正器到期 → 股价回归中性
```

### 9.2 信誉值

玩家有一个持续追踪的**信誉值**，范围 0~100，初始 50：

| 信誉区间 | 称号 | 效果 |
|---------|------|------|
| 80~100 | 权威财经媒体人 | 文章影响力 ×1.3，解锁主流媒体渠道 |
| 60~79 | 知名财经博主 | 文章影响力 ×1.1，解锁主流媒体渠道 |
| 40~59 | 普通自媒体 | 无加成无惩罚 |
| 20~39 | 争议性写手 | 文章可信度 ×0.8，主流媒体渠道关闭 |
| 0~19 | 阴谋论博主 | 主流媒体渠道关闭，但解锁特殊能力"邪典效应"：阴谋论文章影响力 ×2（极端粉丝群体） |

信誉值变动：
- 发表**高可信度**文章且未被打假 → +3
- 发表文章被事实核查打假 → -10
- 发表的文章预测与实际走势一致 → +5（"事后诸葛亮"加成）
- 每回合自然恢复 +1（公众遗忘）

## 12. 黑色幽默元素

贯穿整个游戏的讽刺和荒诞要素：

### 12.1 反讽事件（特殊环境牌）

从突发事件牌堆中低概率出现的特殊事件：

| 事件名称 | 效果 |
|---------|------|
| "你编的假消息居然成真了" | 最近一篇被打假的文章效果恢复，信誉 +5 |
| "认真写的分析被当成软文" | 最近一篇高可信度文章影响力减半 |
| "竞争对手抄袭你的假新闻" | 对应股票情绪效果再次触发（免费加倍） |
| "监管约谈但调查员也在炒股" | 收到罚款通知但罚款金额以股票形式支付给你 |
| "AI 生成的分析报告满天飞" | 本回合所有文章影响力 ×0.5（信息过载） |
| "股评人跳槽当了脱口秀演员" | 随机一位"专家"观点卡从牌堆中移除 |

### 12.2 卡牌风味文字示例

**素材卡**：
- 谣言卡「据不愿透露姓名的保洁阿姨称」：可信度 1，影响力 4
- 数据卡「第三方审计报告（第47页脚注）」：可信度 5，影响力 1
- 爆料卡「CEO的前妻在朋友圈发的截图」：可信度 2，影响力 5
- 观点卡「从未看对过的首席分析师最新研判」：可信度 3，影响力 3

**写作方法卡**：
- 「震惊体」：你的文章标题必须以感叹号结尾！！！（影响力 ×1.5，可信度 ×0.5）
- 「AI代笔」：一分钟出稿，质量随缘（精力消耗减半，可信度 -2）
- 「学术黑话」：没人看得懂但显得很专业（可信度 +2，影响力 -1）

## 14. 待定与开放问题

以下设计点在 v1 中暂不确定，需要通过原型测试验证：

### 14.1 待设计：框架层

以下设计点尚未确定规则，需要在下一轮设计讨论中闭合：

1. **信息可见性规则**：玩家能看到对手的哪些信息？需要逐项确认：
   - 对手的现金和持仓是否公开？（公开 = 能推测对手意图；隐藏 = 更多猜测空间）
   - 对手手牌数量是否可见？（影响打假决策——知道他手里有 8 张卡还是 2 张卡）
   - 已发表文章的详细属性（影响力、可信度数值）是否公开？（需要公开才能判断是否值得打假）
   - 对手的信誉值是否公开？
   - 对手在黑市的买卖行为是否公开？

2. **手牌上限**：素材卡持有是否有上限？如果没有，玩家可能一直囤卡不出手，等待最佳时机一次性打出。需要决定：
   - 是否设置手牌上限？（如最多 8 张）
   - 超出上限时如何处理？（弃牌 / 强制卖入黑市 / 不再抽牌）
   - 手牌上限是否因角色而异？

3. **胜负判定细节**：当前规则为"20 回合内资产总额达到目标值"，需要补充边界情况：
   - 如果无人达标，最高资产者胜还是全员失败？
   - 多人同时达标如何排名？
   - 是否存在淘汰机制？（如资产归零是否出局）
   - 是否有提前结束条件？（如某人资产远超目标，游戏是否继续）

4. **玩家回合顺序**：多人游戏中行动顺序直接影响策略，需要确认：
   - 回合内玩家行动顺序如何确定？（固定 / 每回合轮换 / 随机 / 竞拍）
   - 先行动者暴露意图但抢占先机，后行动者有信息优势——这个不对称是有意设计还是需要补偿？
   - PUBLISH 和 CLARIFY 阶段的顺序尤其关键——是所有玩家同时发表，还是按顺序发表？

5. **游戏开局流程**：从"打开游戏"到"第一回合开始"之间的流程需要定义：
   - 角色选择顺序如何确定？（随机 / 房主先选 / 同时盲选）
   - 选角色后是否有初始抽牌阶段？（抽几张素材卡作为起始手牌）
   - 是否有初始建仓机会？（在第一回合前买入股票）
   - 是否展示所有玩家的角色选择？（公开 = 开局就有针对性；隐藏 = 前几回合需要试探）

### 14.2 待定：数值层

以下数值需要通过原型测试（playtest）来调整：

1. **数值平衡**：素材卡的可信度/影响力范围、波动系数、初始资金等需要大量调参
2. **做空机制**：当前只支持买入持有，做空可以让利空文章策略更有趣，但会增加新手理解成本
3. **多局 Meta 进程**：单局 20 回合结束后，是否有跨局解锁内容（如解锁新角色）？
4. **素材卡数量**：初版需要多少张不同的素材卡才能保证足够的组合多样性？初步估算每种类型 8~12 张，共约 32~48 张
5. **环境牌数量**：日程事件和突发事件各需要多少张？
6. **角色平衡**：6 个角色之间的强度平衡需要大量 playtest 验证
7. **黑市数值**：基准价、随机范围、系统上架频率需要通过测试调整

## 15. 情报黑市系统

### 15.1 概述

情报黑市是一个系统做市商驱动的素材卡交易市场。玩家可以在黑市买入或卖出素材卡，无需等待其他玩家配对。黑市解决了两个核心问题：为素材卡提供除写文章之外的第二种用途（变现），以及为玩家提供定向获取素材的额外渠道。

### 15.2 定价机制

每张素材卡根据稀有度确定**基准价**，在卡牌生成时同时确定买入价和卖出价：

```
定价公式：
  基准价 = 稀有度阶梯价（见下表）
  黑市售价（玩家买入）= 基准价 + 随机值
  玩家卖出价（玩家收入）= 基准价 - 随机值

  差价 = 黑市的利润（系统吃掉，模拟做市商价差 spread）
```

| 稀有度 | 基准价（暂定） | 买入价范围 | 卖出价范围 |
|-------|--------------|-----------|-----------|
| 普通 COMMON | 200 | 220~300 | 100~180 |
| 稀有 UNCOMMON | 500 | 550~700 | 300~450 |
| 罕见 RARE | 1000 | 1100~1400 | 600~900 |

卡牌的售价在**生成时就确定**，不会随市场变化。玩家卖出的卡进入黑市后，按系统定价规则重新定价。

### 15.3 黑市库存

**系统自动上架：**

```
系统上架规则：
  · 每回合系统最多上架 3 张素材卡
  · 稀有度按概率决定：普通 60% / 稀有 30% / 罕见 10%
  · 素材类型、行业、立场随机
  · 当黑市内系统上架的卡累积达到 5 张时，系统停止注入
  · 玩家购买系统卡后，若库存低于 5 张，下回合系统继续补充
```

**玩家卖入：**

```
玩家卖出规则：
  · 玩家可随时将手牌卖入黑市
  · 卖入数量无上限
  · 卖出价在卡牌生成时已确定
  · 卖入的卡立即进入黑市库存，其他玩家可购买
```

### 15.4 信息展示

玩家在黑市中可以看到每张卡的以下信息：

```
可见信息：
  · 素材类型（数据 / 爆料 / 谣言 / 观点）
  · 关联行业（科技 / 金融 / 能源 / 医药 / 宏观）
  · 价格

不可见信息：
  · 立场（利好 / 利空）
  · 具体数值（可信度 / 影响力）
```

设计意图：玩家知道"这是一张科技行业的爆料卡，售价 650 元"，但不知道是利好还是利空。这保留了信息博弈——"我要不要赌一把？"同时也创造了社交推理空间——"他为什么把这张卡卖了？是不需要，还是故意投放利空信息？"

### 15.5 交易规则

```
· 买入素材卡：消耗现金，不消耗行动值
· 卖出素材卡：获得现金，不消耗行动值
· 每回合买卖次数：暂不限制
· 时机：GATHER 阶段可执行（与取材同阶段）
```

## 16. 文章打假系统（玩家对抗）

### 16.1 概述

打假就是写文章——玩家通过撰写一篇**反驳文章**来攻击对手已发表的文章。这不是系统自动判定，而是玩家的主动对抗行为。反驳文章使用与普通文章完全相同的合成系统，唯一的区别是目标从"股票"变为"对手的文章"。

```
文章目标类型：
  · 目标是股票 → 影响股价情绪（现有逻辑）
  · 目标是对手的文章 → 与对手文章进行影响力比对（打假逻辑）
```

### 16.2 澄清窗口

每篇文章发表后，在 CLARIFY 阶段触发澄清窗口：

```
澄清窗口流程：
  1. 文章发表，所有玩家看到文章的公开信息
  2. 其他玩家选择"打假"或"跳过"
  3. 选择打假的玩家进入合成界面，用手牌素材合成反驳文章
  4. 反驳文章必须与被打假文章立场相反（利好文章只能用利空文章打）
  5. 所有反驳文章提交后，进入结算
```

### 16.3 打假结算

```
打假方总影响力 = 所有反驳文章影响力之和（多人可叠加攻击同一篇文章）

情况一：打假成功（打假方总影响力 > 被打假文章影响力）
  → 被打假文章的情绪修改器移除
  → 溢出影响力 = (打假方总影响力 - 被打假文章影响力) × 2
  → 溢出部分以反向立场附加到目标股票上（作为新的情绪修改器）
  → 溢出修改器持续回合数 = 反驳文章中可信度最高者的可信度值

情况二：打假失败（打假方总影响力 ≤ 被打假文章影响力）
  → 被打假文章影响力减去打假方总影响力，继续生效
  → 反驳文章直接消耗，不产生任何效果（素材和行动值白费）
```

### 16.4 完整示例

```
场景：A 发表利好文章，影响力 10，目标星海科技

澄清窗口：
  B 选择打假，合成利空反驳文章，影响力 7
  C 选择打假，合成利空反驳文章，影响力 5
  D 选择跳过

结算：
  打假方总影响力 = 7 + 5 = 12
  12 > 10 → 打假成功！

  · A 的利好情绪修改器被移除
  · 溢出 = (12 - 10) × 2 = 4 点利空情绪附加到星海科技
  · A 的操控计划被反噬——不仅文章失效，股价还反向下跌

反面场景：如果 C 没有参与打假
  打假方总影响力 = 7
  7 < 10 → 打假失败

  · A 的文章影响力从 10 降为 3，继续生效
  · B 的反驳文章消耗掉，素材白费
  · B 承担了风险但未能翻转局面
```

### 16.5 打假的战略深度

打假机制围绕素材卡的多用途设计产生了多层博弈：

**攻击方（打假者）的决策：**
- 我手里的素材是写文章赚钱，还是留着打假别人？
- 我一个人影响力不够，要不要联合其他玩家？但联合意味着暴露我的持仓意图
- 我现在不打，等对手继续加仓再打，他的损失更大——但风险是等太久文章效果已经生效太多回合

**防守方（被打假者）的决策：**
- 我要不要发这篇高影响力文章？对手手里可能有反驳素材
- 我可以先发一篇小文章试探，看看有没有人打假，再决定要不要发大的
- 我可以虚张声势："我劝你三思"——即使我手里没有相关素材

**旁观者的决策：**
- 两个人在打架，我要不要趁机在他们忽略的股票上操作？
- 我要不要加入打假方，趁机打压这只股票然后低价买入？

### 16.6 打假与系统事实核查的关系

原有的系统事实核查（9.1 节）与玩家打假并存，但触发时机不同：

```
PUBLISH  → 文章发表
CLARIFY  → 玩家打假窗口（主动对抗）
MARKET_REACT → 系统事实核查（被动概率判定）

一篇文章可能经历两重考验：
  1. 先在 CLARIFY 阶段被玩家打假（可能被削弱或移除）
  2. 如果存活，再在 MARKET_REACT 阶段面临系统事实核查
```

这意味着低可信度文章面临双重风险——既可能被玩家锤，又可能被系统打假。高风险高回报的设计初衷得到了加强。

## 17. 角色系统

### 17.1 概述

每位玩家在游戏开始前选择一个角色。角色决定了玩家的**被动技能、主动技能、专属写作方法卡和初始资源**。不同角色拥有截然不同的玩法路线，角色选择即是策略构建的起点。

角色选择规则：
- 每局游戏中同一角色只能被一位玩家选择（不可重复）
- 初版提供 6 个角色，支持 2~6 人游戏
- 选人阶段可产生博弈——"他选了营销号主，那我选调查记者专门锤他"

### 17.2 角色数据结构

```
Character {
    id: StringName            # 唯一标识
    name: String              # 角色名
    title: String             # 头衔/身份描述
    passive_skill: Skill      # 被动技能（全局生效，无需激活）
    active_skill: Skill       # 主动技能（消耗资源，每回合限用一次）
    writing_methods: Array[WritingMethodCard]  # 专属写作方法卡（4张）
    starting_cash: int        # 初始现金（标准 10000）
    starting_reputation: int  # 初始信誉（标准 50）
    starting_connections: int # 初始人脉（标准 3）
    flavor_text: String       # 人物介绍（黑色幽默风格）
}

Skill {
    id: StringName
    name: String
    description: String
    cost: Dictionary          # 消耗资源（被动技能为空）
    cooldown: int             # 冷却回合数（0=每回合可用）
    is_passive: bool          # 是否为被动技能
}
```

注：所有角色额外持有 1 张通用基础卡「平铺直叙」，不计入专属写作方法卡数量。

### 17.3 角色一览

---

#### 「数据分析师」—— 前券商研究员

> "市场上 90% 的人在猜，我在算。"

**玩法路线**：信息优势型。靠精准预判日程事件提前布局，用高可信度文章稳定获利。

| 项目 | 内容 |
|------|------|
| 被动技能 | 「内部渠道」：每回合免费窥探一次日程事件（不消耗人脉） |
| 主动技能 | 「量化模型」：消耗 1 行动值，查看一只股票当前所有活跃情绪修改器的详细数值 |
| 专属写作方法 | 数据可视化、交叉验证、严谨论证、长期追踪 |
| 初始现金 | 8000（低于标准） |
| 初始信誉 | 50 |
| 初始人脉 | 3 |

**设计意图**：起始资金少，但信息优势强大——每回合免费窥探意味着每个周期都能提前知道事件好坏，精准建仓。写作方法偏高可信度路线，文章效果持续时间长、被打假风险低，适合稳健型玩家。

---

#### 「营销号主」—— 十万+爆文制造机

> "什么叫真相？阅读量就是真相。"

**玩法路线**：高影响力高风险型。文章影响力天然增强，但更容易被打假和事实核查。适合激进操盘，快进快出。

| 项目 | 内容 |
|------|------|
| 被动技能 | 「流量体质」：所有文章影响力 +2，但事实核查概率额外 +10% |
| 主动技能 | 「蹭热点」：免费引用当前回合任意一个环境事件作为素材（无视行业匹配限制） |
| 专属写作方法 | 标题党、情感渲染、病毒传播、震惊体 |
| 初始现金 | 10000 |
| 初始信誉 | 40（低于标准） |
| 初始人脉 | 3 |

**设计意图**：起始信誉低意味着离"争议性写手"惩罚线很近，但同时也离"阴谋论博主"的邪典效应不远。玩家需要决定是维护信誉走正统路线，还是干脆躺平走邪典路线。写作方法全是高影响力低可信度的组合，天然适合造谣暴拉/暴砸。

---

#### 「调查记者」—— 前财经媒体人

> "你以为你在操控市场？我以为你在给我送素材。"

**玩法路线**：打假反噬型。靠锤对手的文章获利，是其他玩家造谣时的天然克星。

| 项目 | 内容 |
|------|------|
| 被动技能 | 「正义光环」：打假成功时信誉 +10，且反噬溢出倍率从 ×2 提升为 ×3 |
| 主动技能 | 「线人」：消耗 1 人脉，查看某玩家手中素材卡的类型和行业（不看立场和数值） |
| 专属写作方法 | 深喉网络、匿名化处理、严谨论证、交叉验证 |
| 初始现金 | 10000 |
| 初始信誉 | 50 |
| 初始人脉 | 5（高于标准） |

**设计意图**：高人脉让调查记者既能窥探日程又能定向取材又能查看对手手牌，信息获取能力全角色最强。打假溢出 ×3 意味着成功打假的收益极高——但前提是有人在造谣。如果其他玩家都在稳健写文章，调查记者的被动就浪费了。角色的强度取决于对手的玩法。

---

#### 「庄家」—— 神秘资本玩家

> "散户看K线，我画K线。"

**玩法路线**：交易优势型。更多的交易次数和启动资金，可以通过频繁买卖和微操股价来获利。

| 项目 | 内容 |
|------|------|
| 被动技能 | 「大资金」：每回合交易次数 5 次（标准 3 次），买入无最小手数限制 |
| 主动技能 | 「对倒」：消耗 500 元，对一只股票施加一个 1 回合的微小情绪修改器（+2 或 -2），不需要写文章 |
| 专属写作方法 | 春秋笔法、断章取义、危机公关、连续剧 |
| 初始现金 | 15000（高于标准） |
| 初始信誉 | 50 |
| 初始人脉 | 3 |

**设计意图**：庄家是唯一一个不完全依赖文章就能影响股价的角色——「对倒」提供了一个小幅但确定的股价推力。更多的交易次数让庄家可以分批建仓/出货，降低被对手一波打假的风险。写作方法偏操控类，「春秋笔法」可以反转素材立场，「危机公关」可以反转情绪修改器，适合复杂操盘。

---

#### 「情报贩子」—— 地下信息掮客

> "在这个市场上，消息比钱值钱。"

**玩法路线**：黑市操控型。在情报黑市中以优惠价格买卖素材卡，通过信息差获利。

| 项目 | 内容 |
|------|------|
| 被动技能 | 「关系网」：黑市买入价 -30%，卖出价 +20% |
| 主动技能 | 「定制情报」：消耗 1 行动值，从黑市购买时可额外看到卡牌的立场（利好/利空） |
| 专属写作方法 | 匿名化处理、春秋笔法、深喉网络、标题党 |
| 初始现金 | 10000 |
| 初始信誉 | 50 |
| 初始人脉 | 3 |
| 特殊 | 每回合额外获取 1 张素材卡 |

**设计意图**：情报贩子的核心优势是**素材卡流转效率**——买得便宜、卖得贵、看得清、抽得多。他可以低价收购素材卖高价赚差价，也可以精准购买需要的立场卡来组合强力文章。写作方法混搭攻防，「匿名化处理」降低打假风险，「春秋笔法」反转立场创造灵活性。

---

#### 「阴谋论者」—— 民间"股神"

> "你们笑我疯，我笑你们看不穿。"

**玩法路线**：低信誉邪典型。主动放弃信誉，拥抱邪典效应，以极高影响力的文章暴力操控市场。

| 项目 | 内容 |
|------|------|
| 被动技能 | 「邪典教主」：信誉低于 20 时所有文章影响力 ×2；信誉永远不会自然恢复（每回合不 +1） |
| 主动技能 | 「洗版」：消耗 2 行动值，目标股票上所有活跃情绪修改器持续时间 -1（加速过期，制造混乱） |
| 专属写作方法 | 情感渲染、病毒传播、震惊体、连续剧 |
| 初始现金 | 10000 |
| 初始信誉 | 20（远低于标准，起步即在邪典边缘） |
| 初始人脉 | 3 |

**设计意图**：最极端的角色。起步信誉 20 且不会自然恢复，意味着玩家几乎必然走邪典路线——一旦被打假或事实核查信誉降到 20 以下，影响力直接翻倍。配合「病毒传播」（影响全行业）和「连续剧」（连续发文影响力 ×2），可以在短时间内制造巨大的市场波动。代价是极高的打假风险——调查记者和数据分析师是他的天敌。「洗版」主动技能用来清除市场上的既有修改器，为自己的操控扫清障碍。

### 17.4 角色间的克制关系

角色之间存在自然的克制和协同关系，这鼓励玩家根据对手的角色选择来调整策略：

```
克制关系：
  调查记者 克制 营销号主/阴谋论者（专门打假高风险文章）
  数据分析师 克制 庄家（信息透明化削弱庄家的隐蔽操盘）
  阴谋论者 克制 情报贩子（极高影响力文章让黑市信息价值降低）
  庄家 克制 调查记者（不依赖文章就能推动股价，打假无从下手）
  情报贩子 克制 数据分析师（更多素材来源，信息获取渠道更广）
  营销号主 克制 庄家（影响力碾压庄家的微操，大力出奇迹）

注：克制关系是软性的，不是绝对的。实际对局中玩家的操作和决策远比角色选择重要。
```
