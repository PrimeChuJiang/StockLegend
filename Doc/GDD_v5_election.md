# 竞选大师 — 核心规则 v5

> 黑色幽默政治竞选卡牌涂地游戏。
> 核心玩法：手牌叠放合成形状，重叠部分涂地生效。
> 美式互攻桌游，直接对抗。

---

## 1. 一句话描述

你是一个不择手段的竞选经理，通过发表文章（叠放手牌合成形状）来涂地争夺选民。10回合后数格子，占地多者胜。

## 3. 地图

10×10 的格子棋盘，每个格子有两个属性：

| 属性 | 说明 |
|------|------|
| 归属 | 中立 / 玩家A / 玩家B |
| 忠诚度 | 整数 ≥ 0，初始为 0 |

初始状态：所有格子均为中立。

---

## 4. 核心操作：发表文章

### 4.1 操作流程

1. 消耗 **1 行动点**
2. 从手牌中选择 **2~4 张卡牌**
3. 将所有卡牌**叠放**，玩家自由调整每张卡的相对位置
4. 只有**重叠部分**的格子实际生效
5. 选择投放到棋盘上的位置（全图任意位置，无限制）

### 4.2 重叠规则

每张卡牌有一个形状模板（若干格子组成的图案）。叠放时，玩家可以对每张卡牌进行以下操作：

- **旋转**：0°/90°/180°/270° 自由旋转
- **对齐位置**：自由指定每张卡的偏移位置，决定从哪里开始重叠

叠放后，多张卡牌的形状可能在某些格子上重叠：

- 2张卡叠放：重叠格最大层数 = 2
- 3张卡叠放：重叠格最大层数 = 3
- 4张卡叠放：重叠格最大层数 = 4

**只有至少被2张卡覆盖的格子才算"重叠"，才会生效。** 仅被1张卡覆盖的格子无效果。

### 4.3 威力值计算

生效格子的基础威力值 = **重叠层数 - 1**：

| 重叠层数 | 威力值 | 说明 |
|---------|--------|------|
| 1张 | 不生效 | 未达到重叠阈值 |
| 2张重叠 | 1 | 最低生效 |
| 3张重叠 | 2 | |
| 4张重叠 | 3 | 最大值（最多4张叠放） |

特殊卡牌可能修改最终威力值（如乘数、额外加成等），系统需预留接口支持。

### 4.4 格子结算

根据威力值对目标格子生效：

| 目标格状态 | 效果 |
|-----------|------|
| 中立格 | 翻色为己方，忠诚度 = 威力值 |
| 己方格 | 忠诚度 + 威力值 |
| 敌方格 | 忠诚度 - 威力值，归零后继续打则翻色（剩余威力成为己方忠诚度） |

### 4.5 示例

```
卡A：              卡B：              卡C：
■ ■ ■              ■ ■                ■
■                  ■ ■                ■
                   ■                  ■

三张叠放（★3 = 三张重叠，★2 = 两张重叠，· = 仅1张不生效）：

·  ★3 ★2
·  ★2
   ·

★3 格：威力 = 3-1 = 2
★2 格：威力 = 2-1 = 1
·  格：不生效

→ 如果目标全是中立格：
  ★3 格翻色为己方，忠诚度2
  ★2 格翻色为己方，忠诚度1
```

## 5. 回合流程

```
┌─────────────────────────────────────────────┐
│ 1. 抽牌阶段                                  │
│    · 首回合：双方各抽 5 张                     │
│    · 之后每回合：双方各抽 2 张                  │
├─────────────────────────────────────────────┤
│ 2. 玩家A行动阶段                              │
│    · 3 行动点                                 │
│    · 发表文章：1行动点 + 2~4张手牌              │
│    · 用完或主动结束                            │
├─────────────────────────────────────────────┤
│ 3. 玩家B行动阶段                              │
│    · 同上                                     │
├─────────────────────────────────────────────┤
│ 4. 结算阶段                                  │
│    · 超出手牌上限（5张）则弃牌至5张             │
│    · 无忠诚度衰减                              │
└─────────────────────────────────────────────┘
```

2. 占领格子多的一方获胜
3. 平局规则：待定

---

## 9. 策略空间

核心博弈：**面积 vs 强度**

- 最大重叠（少量格子，高威力）→ 集中火力，攻坚高忠诚度区域
- 最小重叠（大面积，低威力）→ 薄涂铺开，快速抢地盘
- 同样的手牌，不同叠放方式 = 完全不同的战术

## 11. 与 v4 的主要变化

| v4 | v5 |
|----|-----|
| 5个选区多线作战 | 10×10 棋盘涂地 |
| 2张素材合成广告（固定类型表） | 2~4张卡牌叠放合成形状（自由重叠） |
| 数值结算（拉票/攻击系数） | 空间结算（重叠层数 = 威力） |
| 议题匹配加成 | 移除议题概念 |
| 选区赢者通吃 | 逐格计分 |
